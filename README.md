## Projects

### JARVIS-style AI Assistant *(Experimental Project)*

One of my experiments i did a while ago was building a **local AI assistant system** that combines several tools together.

This project was mainly about learning how different AI tools interact.  
I built it with some help from **Anthropic's Claude**, which helped me understand parts of the implementation.

Technologies explored:

- speech recognition using **Whisper**
- computer vision using **OpenCV + YOLO**
- local language models through **Ollama**
- Python-based AI pipelines

The goal wasn't to build something perfect — it was to **learn how modern AI systems can be stitched together**.
